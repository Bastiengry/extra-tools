Le fichier incredy-realm-docker.json permet de charger un realm et configuration du client

Il faut encore ensuite créer à la main :
- un groupe
- un utilisateur :
    - ID, email, Nom, Prénom
    - Ensuite, il faut ajout un mot de passe (ne pas activer la demande de changement à la première connexion)
    - Associer l'utilisateur au groupe
